# 金融数据与文献分析平台

## A Desktop Application for Financial Data Analysis and Document Processing

* 技术栈: Python · Tkinter · Matplotlib · Tushare · Jieba · Requests

* 课程名称: 数据挖掘与分析
* 我的角色: 独立开发者 (课程项目)

## 📊 项目概述

这是一个基于Python Tkinter的桌面应用程序，集成了金融数据获取、股票分析、文档处理和文本挖掘功能。项目旨在通过统一的可视化界面，帮助用户获取和分析股票市场数据，同时处理金融文献内容，实现了数据处理与分析的全流程实践。

### 金融数据与文献分析平台
这是一个基于Tkinter的桌面应用程序，主要完成以下两个核心任务：
1. 金融股票数据分析
* 数据获取：通过 Tushare金融数据接口，根据用户输入的股票代码和日期范围，获取股票的历史行情数据（开盘价、收盘价、最高价、最低价、成交量）。
* 股价走势图：绘制多指标（开盘、收盘、最高、最低价）的走势曲线，并叠加成交量柱状图。
* 指标相关性热力图：计算并可视化股价各技术指标（如开盘价与收盘价）之间的相关系数矩阵。

2. 金融文献文本挖掘
* 文献获取：从指定的阿里巴巴集团投资者关系页面，爬取并下载年度报告（PDF格式）。
* 文本分析与可视化：
* 词云生成：对PDF报告进行文本提取、中文分词和词频统计，生成关键词词云图。
* 高频词统计：统计并可视化文档中出现频率最高的前10个词汇的横向柱状图。

### 社交媒体数据情感分析工具
这是一个同样基于Tkinter的桌面应用，其核心任务聚焦于社交媒体数据的情感分析。
核心任务为多平台舆情情感分析：
* 数据获取：允许用户输入关键词，从微博、知乎、豆瓣等多个社交媒体平台爬取相关的帖子、问答或评论内容。
* 情感分析：使用 SnowNLP库对爬取的中文文本进行情感倾向分析，为每一条内容计算情感得分（通常0为负面，1为正面）。
* 情感分布图：通过饼图或柱状图，直观展示整体评论中“正面”、“中性”、“负面”情感的占比分布。
* 情感趋势图：如果数据带时间戳，可以分析情感随时间的变化趋势。
* 高频词与词云：对文本进行分词和词频统计，生成词云图，并展示正面/负面评价中的高频特征词。
